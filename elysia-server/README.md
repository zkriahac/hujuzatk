# Elysia Backend Server

This is a minimal backend server using the [Elysia](https://elysiajs.com) framework (Node.js/Bun).

## Getting Started

### Prerequisites
- [Bun](https://bun.sh/) installed (recommended for best performance)

### Install dependencies
```bash
bun install
```

### Run the server
```bash
bun run index.ts
```

The server will be available at http://localhost:4000

---

## Next Steps
- Add your API routes and business logic to `index.ts` or split into modules.
- Integrate with your database (e.g., Prisma, PostgreSQL) as needed.
- Update your frontend to use this new backend endpoint.
